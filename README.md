# Task To-Do

A full-stack To-Do application built with Next.js App Router, TypeScript, and Prisma.

## Features
- Task CRUD operations
- Soft delete
- RESTful API

## Tech Stack
- Next.js
- Prisma
- SQLite
- 

## Running Locally
```sh
    git clone https://github.com/your_username/agit-todo-app.git
    npm install
    npx prisma migrate dev
    npm run dev
```
